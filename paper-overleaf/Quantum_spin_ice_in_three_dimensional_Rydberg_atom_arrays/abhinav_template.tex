We thank you once again for the consideration given to the manuscript. We also thank all the Referees for their time.

The point brought up by Referee D, namely that we mention a change to a previous version but the version in question is not public, is entirely fair. To rectify this, we have changed that sentence on page 4 to cite the first arXiv version (v1) of the manuscript, by saying instead:
\wesaid{Stronger Lieb-Robinson bounds can increase $t_\mathrm{easy}$,
and faster state-transfer can reduce $t_\mathrm{hard}$, as evidenced by the
improvement over the first version of this manuscript [59] due to results from Ref. [44].}

We have also attached a file that highlights these changes, with additions in blue and deletions in red. The only other changes highlighted in that file are cosmetic changes in the references.

Thanking you, \\
Sincerely, \\
